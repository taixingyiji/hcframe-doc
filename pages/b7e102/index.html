如何设置以及启动项目

# 如何设置以及启动项目

# 安装依赖

yarn install
1

# 启动本地开发环境（自带热启动）

yarn serve
1

# 构建生产环境 (自带压缩)

yarn build:prod
1

# 构建阶段性生产环境 (自带压缩)

yarn build:stage
1

# 代码格式检查以及自动修复

yarn lint
1

# 运行单元测试

yarn test:unit
1

# 运行端对端测试

yarn test:e2e
1

# 自动生成 svg 组件

yarn run svg
1

# 自定义 Vue 配置
